# 九安智能采集 V0.9.11

本版本升级为角色独立配置版。

## 角色配置

安装前可修改：

```text
config/collector-role.config.json
```

默认启用三个角色：

- `1` / `customer_report`：客服-数据报表
- `2` / `finance_withdraw`：财务-提现账单
- `3` / `finance_tax`：财务-涉税账单

示例：只启用客服和涉税：

```json
{
  "enabledRoles": [1, 3]
}
```

## 自动采集计划

每个角色独立配置自动采集计划，支持：

- 按天：每天
- 按天：工作日
- 按天：指定周几
- 按月：每个月第几号

平台探活保持轻量首页刷新，采集中会自动跳过探活，避免打断采集。

## 当前接入状态

- 客服-数据报表：沿用现有微信小店、抖店、拼多多采集链路。
- 财务-提现账单：已完成角色入口和独立计划配置，采集接口预留。
- 财务-涉税账单：已完成角色入口和独立计划配置，采集接口预留。
